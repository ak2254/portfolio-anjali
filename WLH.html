PROJECT DETAILS

THE PROJECT OVERVIEW

The project presents the combination of robust frontend representation of the company’s brand as well as an easily manipulated CMS backend for the company to be able to make changes in the text as well the images without requiring to go into the code itself. The entirety of the site is implemented utilizing PHP, bootstrap, HTML/CSS and custom fields while keeping the customer's style requirements in mind.

STYLING DETAIL

THE SITE FOLLOWS A STRICT TWO COLOR COMBINATION OVERALL:

The custom designed Wordpress site includes:

  • Flow of the site follows dark grey and white background as per the section change to draw the attention to the new section
  • Each image is precisely sized according to the style guideline utilizing Adobe
  • All the text/tag follow a strict template throughout the site to be consistent
  • Buttons are presented in the same color to represent its primary utilization

IMPLEMENTATION DETAIL

CUSTOM FILED UTILIZATION:


IN DETAIL:

  • Each page on the site is implemented utilizing custom fields that are used to echo the php content.
    • All the content including the text as well as the images are presented in a way that the customer is able to not only change the content dynamically but also add a row of components without requiring access to the code.
  • The image presented shows one of the project detail page on the site.
    • These pages are the most important part of the company's utilization of the dynamic backend. As per the implementation, PHP while loops are utilized to present the custom fields.
    • Goal accomplished: The customer is easily able to add the projects as well its content and the gallery images by dynamically dumping the images.

IMPLEMENTATION DETAIL

WPforms:

As per the forms, they are customized to work with mailchimp's API to store the data. They are styled while overriding the basic WPforms styling using CSS.
