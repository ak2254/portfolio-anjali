PROJECT DETAILS

THE APP OVERVIEW

This project requires you to make a basic banking application that follows the requirements below. It consists of a list of accounts and their current balance (stored in Redux). A list of transactions for all modifications made to the balance of each account (also stored in redux). Lastly, it consists of ways to deposit and withdraw money, as well as add and delete existing accounts.

REQUIREMENTS DETAIL

Display a list of all accounts and all transactions:

All the overview of the accounts are displayed on the home page of the application. The overview of all the transaction taken place in the application are also displayed in the list/table view in the transactions page. As per the retrial of the accounts and the transactions, they are retrieved from an API (JSON Test Server) and stored in Redux State.

REQUIREMENTS DETAIL

An accounts Detail page:

These pages are overall designed to display each account's information, balance, and transaction history.
IN DETAIL:

  • Account'S ID and Information is retrieved from the URL
    Button to Delete and Edit the account
  • The page also has the ability to add and deposit money to the account. This also creates a new transaction record as well as updates the balance for the account everywhere it’s displayed.

REQUIREMENTS DETAIL

Add and Delete Account Pages:


IN DETAIL:

  • Add Account Page:
    • Contains a form with the ability to create a new account with an initial balance
  • Delete Account Page:
    • Consist of a form with the ability to delete an account based on its ID.
