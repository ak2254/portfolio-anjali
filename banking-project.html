PROJECT DETAILS

THE APP OVERVIEW

This project requires you to make a basic banking application that follows requirements bellow. It consists a list of accounts and their current balance (stored in Redux). A list of transactions for all modifications made to the balance of each account (also stored in redux). Lastly, it consists of ways to deposit and withdraw money, as well as add and delete existing accounts.

REQUIREMENTS DETAIL

Display a list of all accounts and all transactions:

All the overview of the accounts are Displayed on the home page of the application. The overview of all the transaction taken place in the application are also displayed in the list/table view in the transactions page. As per the retrial of the accounts and the transactions, they are retrieved from an API (JSON Test Server) and stored in Redux State.
