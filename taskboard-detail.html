PROJECT DETAILS

THE APP OVERVIEW

This is a simple project management application with an basic set of features
INCLUDING:
A board view where the user can view all of the tasks in columns relating to their status
A list view to view and filter all tasks at once
an add form to be able to add new tasks to the project.

REQUIREMENTS DETAIL

Task Board Component:

This component allows the user to see all of the tasks sorted into columns by their status (to do, in progress, review, and done). The component displays all tasks in a card like UI, sorted into their appropriate columns. The Board also has the ability to move a card/task from one column to another. The task stays in it’s new column until a hard refresh.

REQUIREMENTS DETAIL

Task List Component:

The task list component display all the tasks in a table with all of its data displayed. The list also has the capability of being filtered by Column/Status and Type as well as sorted by all of the task’s properties.

REQUIREMENTS DETAIL

Add Task Component:

The Add Task Form component is able to mock adding a task to the database. The form submitted updates the state variables on the App component so any other component in the application can see the new Task until a hard refresh. Upon Submitting the form, the user is taken to the Board view where he/she views newly added Task.
